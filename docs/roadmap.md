# Jumping Monkeys — Roadmap (4 ספרינטים)

## Sprint 1 — Foundation + Core Loop (שבוע 1)

### מטרות
- הקמת פרויקט Phaser + TypeScript
- סצנות בסיס: BootScene, GameScene, UIScene
- לולאת משחק בסיסית עובדת

### משימות
- [ ] תשתית build/run
- [ ] קוף עם קפיצה Drag-to-Aim
- [ ] פלטפורמות רגילות
- [ ] מצלמה שעולה עם השחקן
- [ ] תנאי Game Over + Restart

### Deliverable
משחק בסיסי playable עם ניקוד התחלתי.

---

## Sprint 2 — Scoring, Combo, Content (שבוע 2)

### מטרות
- הרחבת לולאת התגמול
- הוספת מגוון בסיסי של אתגרים

### משימות
- [ ] בננות רגילות + זהובות
- [ ] ScoreSystem מלא
- [ ] ComboSystem עם timeout
- [ ] פלטפורמות נעות
- [ ] UI: score, combo, pause

### Deliverable
Core loop ממכר וברור עם פידבק מיידי.

---

## Sprint 3 — Difficulty + Game Feel (שבוע 3)

### מטרות
- העלאת עומק ושיפור תחושת המשחק

### משימות
- [ ] DifficultySystem לפי זמן/גובה
- [ ] פלטפורמות שבירות
- [ ] particles באיסוף
- [ ] camera shake באירועים
- [ ] שיפור balancing של physics

### Deliverable
משחק עם קצב קושי הדרגתי ותחושת פעולה טובה.

---

## Sprint 4 — Polish + Release Candidate (שבוע 4)

### מטרות
- יציבות, ביצועים, והכנה להפצה

### משימות
- [ ] שמירת best score ב-localStorage
- [ ] אופטימיזציה (cleanup/pooling)
- [ ] QA checklist מלאה
- [ ] תיקוני UX למסכי Game Over/Pause
- [ ] build דפדפן מוכן לפרסום

### Deliverable
גרסת MVP יציבה ומוכנה לשיתוף ראשוני.

---

## KPI Tracking (Post-MVP)

- Average session time
- מספר משחקים לסשן
- התפלגות ניקוד שיא
- Day-1 retention (אם יתווסף analytics)
